Ce projet est une une application web de blog de voyage développée en HTML, Tailwind CSS et JavaScript (Vanilla).
Elle permet d’afficher, d’ajouter et de gérer dynamiquement des cartes de contenu à travers une interface moderne et responsive.

Le projet met l’accent sur la manipulation du DOM, la gestion des événements, l’utilisation de modales et menus contextuels, ainsi que sur une bonne organisation du code sans recourir à des frameworks JavaScript.

Cette application a été réalisée dans un objectif pédagogique afin de renforcer les bases du développement front-end et de comprendre les mécanismes internes des interfaces interactives.
